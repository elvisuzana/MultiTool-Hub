# MultiTool-Hub

Proyek **MultiTool-Hub** adalah platform otomatisasi serbaguna yang mengintegrasikan berbagai teknologi untuk pengelolaan sistem, monitoring, dan otomasi tugas. Proyek ini dirancang sebagai *monorepo* yang mendukung ekosistem backend, frontend, hingga infrastruktur.

## 🛠 Teknologi yang Digunakan
| Modul | Teknologi | Fungsi |
| :--- | :--- | :--- |
| **Backend** | Python | Logika otomasi utama & manajemen node. |
| **Frontend** | TypeScript / JS | Dashboard monitoring & antarmuka. |
| **Web Server** | PHP | API & integrasi web tradisional. |
| **Performance** | C++ | Engine perhitungan performa tinggi. |
| **Infrastructure** | HCL (Terraform) | Konfigurasi cloud & deployment. |
| **UI/UX** | HTML | Struktur halaman web. |

## 🚀 Cara Menjalankan

### 1. Persiapan Backend (Python)
```bash
cd backend
pip install -r requirements.txt
python main.py

2. Frontend & Dashboard (TS/JS)
cd frontend
npm install
npm run dev


### 3. Infrastruktur (Terraform)
cd terraform
terraform init
terraform apply
