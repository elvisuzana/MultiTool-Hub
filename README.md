# MultiTool-Hub
Proyek otomatisasi terpadu yang menggabungkan efisiensi Python dan fleksibilitas Node.js.

## Struktur Direktori
- `/backend`: Berisi script Python untuk logika otomasi utama.
- `/frontend`: Berisi aplikasi Node.js untuk dashboard atau manajemen.

## Cara Install
1. Clone repo: `git clone https://github.com/USERNAME-ANDA/MultiTool-Hub.git`
2. Masuk ke folder backend untuk setup Python.
3. Masuk ke folder frontend untuk setup Node.js.
